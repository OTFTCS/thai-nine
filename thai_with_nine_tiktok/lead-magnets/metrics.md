# Lead Magnet Metrics

Tracking sheet for the Phase 1 manual MVP. One row per launch video. Update weekly.

## Decision rules (pinned)

- **Magnet hits >1.0% view → email** → double down on that topic type next week
- **Magnet hits <0.3%** → topic or CTA is wrong, iterate before continuing
- **Email 1 open rate <40%** → subject line is wrong, A/B test
- **Welcome sequence E2 → E5 drop-off >50%** → rewrite
- **Subscriber count crosses 400** → trigger Phase 3 capacity review

## Phase 2 trigger checklist (review at end of Week 4)

- [ ] Reply volume >30 mins/day across all videos → build ManyChat (Phase 2A)
- [ ] Producing PDFs in Canva taking >1 hr each AND ≥2 weeks of consistent cadence → build PDF generator (Phase 2B)
- [ ] Subscriber count >200 → start planning Phase 3 capacity decision

## Notes on data sources

- **Views, comments, DMs:** TikTok app analytics (pull manually each week).
- **Manual reply count:** Rough self-estimate. The point isn't precision — it's noticing when manual replies become painful.
- **Landing page visits, form submits, PDF downloads, email open/click rates:** MailerLite dashboard (Audience → Reports).
- **Tutoring + course clicks:** Filter by UTM params in MailerLite click tracking. If the tutoring booking page doesn't expose UTM data, track conversions in whatever tool the booking system provides (Calendly, etc.) and cross-reference dates.
- **Tutoring bookings + course enrolments:** Manual count from the booking/course platforms. Correlate by signup date.
