# Thai Course Glossary (Seed)

## Pedagogy terms
- **Primary objective**: One concrete skill learner should gain in a lesson.
- **Micro-quiz**: Short retrieval practice attached to each lesson.
- **Guided practice**: Prompt-pause-reveal segment to force active recall.

## Thai language terms
- **Tone language**: Pitch contour can change word meaning.
- **Live syllable**: Ends in sonorant (n/m/ng/y/w) or long vowel.
- **Dead syllable**: Ends in stop (k/t/p) or short open syllable.
- **Politeness particles**: Thai sentence-final particles (e.g., ครับ, ค่ะ, คะ).

## Production terms
- **READY_TO_RECORD**: Script, visuals, quiz, QA completed and validated.
- **Deterministic path**: Canonical folder path that tools can derive reliably.
- **Runlog**: Timestamped execution notes in `/course/runlogs/latest.md`.
