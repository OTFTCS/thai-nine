# Explanation Research Source Ladder

Use this ladder only for high-risk concepts where a direct translation-plus-usage explanation is still likely to mislead.

## v1 source order

1. Repo-local sources
- `course/research/prerecorded-language-lesson-design.md`
- approved lesson exemplars already in the repo
- current lesson research notes and context

2. Approved external domains
- `https://www.crackinglanguage.com/`
- `https://blog.crackinglanguage.com/`

3. Fallback
- If the approved ladder does not produce a clearly better explanation, stop and teach the concept directly with a translation-first, usage-first explanation.

## Rules

- Do not widen to the general web in v1.
- Use sources to identify misconceptions and useful framing, not to import voice.
- Never copy explanation wording into learner-facing lesson files.
- Keep citations in `explanation-research.md` and distilled notes in `usage-research.md`.
- The final script explanation should be shorter, simpler, and more translation-based than the source note it came from.
