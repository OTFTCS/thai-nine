# Course Pipeline (Immersion Thai with Nine)

For day-to-day operations, use:

- **`course/OPERATIONS.md`** ← single runbook
- **`course/LESSON_PRODUCTION_SOP.md`** ← one-lesson production checklist
- **`course/skill-iteration/README.md`** ← skill one-shot iteration lab for `L002`/`L003`

Stage 3 now produces both the recording deck and a Canva-first export pack.

Core paths:
- Manifest: `course/manifest.yaml`
- Lessons: `course/modules/Mxx/Lyyy/`
- Run log: `course/runlogs/latest.md`
- CLI: `course/tools/pipeline-cli.ts`
- Mission Control: `http://localhost:3000/mission-control`
- Archive of the retired Remotion-first plan: `archive/2026-03-18/course/remotion-first-plan/`
