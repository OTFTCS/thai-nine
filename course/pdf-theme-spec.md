Use this style for every **learner-facing downloadable lesson PDF**.

## Brand
- Header title: **Immersion Thai with Nine**
- Subtitle: `Lesson Notes`
- Color strip: teal `#0ea5a3`

## Typography
- Font family: Thai-capable sans serif (prefer Noto Sans Thai; fallback Tahoma)
- H1 lesson title: 20pt bold
- Section heading: 14pt bold
- Body: 11pt regular
- Table text: 10pt

## Page layout
- A4 portrait
- Margin: 46pt
- Strong whitespace between sections
- Avoid dense paragraph blocks

## Required sections (learner edition)
1. What you will be able to say after this lesson
2. Core phrases (Thai / Transliteration / English)
3. Pronunciation tips (simple and practical)
4. Mini dialogues / role-play
5. Practice drills (self-study)
6. Vocabulary list for review / flashcards
7. Quick recap + memory aids

## Phrase formatting
- Every phrase must appear as triplet:
  - Thai
  - Transliteration (tone-marked)
  - English meaning
- No superscript tone notation allowed.

## Consistency rule
- All lesson PDFs must use this exact layout order and typography stack.
- If a section is missing from source, include a short placeholder line instead of removing the section entirely.
