# Register & Politeness Progression Roadmap

This document defines when and how register/politeness levels are introduced across the course.

## Module-level register targets

| Module | Register target | What learners do |
|---|---|---|
| **M01–M02** | Polite formal only | Always use ครับ/ค่ะ. Formal pronouns (คุณ, ผม/ดิฉัน). No casual forms yet. |
| **M03–M04** | Polite formal + awareness | Continue producing formal. Begin recognizing casual forms when heard (e.g., omitted particles in examples). |
| **M05–M06** | Casual recognition | Understand casual spoken Thai (shortened forms, dropped particles). Produce polite by default, but recognize when Thai speakers don't. |
| **M07–M08** | Casual production (guided) | Begin producing casual register in controlled contexts (with friends, in relaxed scenarios). Know when ครับ/ค่ะ is expected vs optional. |
| **M09–M12** | Register switching | Actively choose formal vs casual based on context. Understand when to use นะ, ล่ะ, สิ as softeners. |
| **M13–M15** | Formal/official register | Introduce formal request language, official Thai, written register awareness. |
| **M16–M18** | Full register awareness | Chat Thai, slang recognition, professional vs casual code-switching. |

## Key register-sensitive forms by module

| Form | Register | First introduced | Production expected |
|---|---|---|---|
| ครับ/ค่ะ/คะ | Polite (mandatory for beginners) | M01-L001 | M01-L001 |
| นะ | Softener (casual-friendly) | M01-L007 | M05+ |
| หรอ/เหรอ | Informal question particle | M06-L001 | M07+ |
| ได้ (as softener) | Casual permission | M01-L007 | M05+ |
| เธอ/กู/มึง | Intimate/rude pronouns | M16+ (recognition only) | Never in course |
| ท่าน | Formal/respectful | M15 | M15+ |

## Rules for script generation

1. Every lesson brief should specify its register target (e.g., "polite formal" or "casual-aware").
2. Roleplay speakers should use register-appropriate language for their roles.
3. When a new register-sensitive form is introduced, explicitly explain the register it belongs to.
4. Do not mix registers without teaching the contrast.
