# M01-L001 — How Thai Works

- **Module:** M01 Thai Script Fundamentals
- **Level:** A0-A1
- **Primary objective:** Learner understands how Thai differs from English (tones, script, particles) and sets correct expectations.
- **Secondary objective:** Use and recognize `สวัสดี`, `ขอบคุณ`, `ครับ/ค่ะ`.
- **Scenario:** First day in Thailand; hearing greetings and polite exchanges.

## Key vocabulary
| Thai | Romanization | English |
|---|---|---|
| สวัสดี | sa-wat-dii | hello/goodbye |
| ขอบคุณ | khop-khun | thank you |
| ครับ | khrap | polite particle (male speaker) |
| ค่ะ | kha | polite particle (female statement) |
| คะ | kha? | polite particle (female question) |

## Common mistakes
1. Treating Thai tones as optional.
2. Assuming Thai script is pictographic.
3. Using `ค่ะ` and `คะ` interchangeably.
